# Videóalapú járműsebesség-becslő rendszer

Ez a projekt egy videóalapú járműkövető alkalmazás és sebesség-becslő.

## Követelmények

- Python 3
- `pip`
- A projekt fájljai:
  - `tracker.py`
  - `requirements.txt`
  - egy bemeneti videófájl (`vehicles.mp4`)

## Telepítés

A szükséges csomagok telepítése:

`pip install -r requirements.txt`

A program futtatása:

`python tracker.py --source_video_path vehicles.mp4 --target_video_path valami.mp4`
